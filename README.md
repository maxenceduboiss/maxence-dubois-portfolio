Portfolio personnel d'étudiant en BUT3 GEII (Génie Électrique et Informatique Industrielle), parcours Systèmes embarqués.

## Structure du projet

```
portfolio/
├── index.html          # Page d'accueil
├── about.html          # À propos
├── experience.html     # Expériences professionnelles
├── projets.html        # Projets techniques
├── skills.html         # Compétences
├── contact.html        # Contact
├── assets/
│   └── logos/          # Logos outils et technologies
└── *.png / *.jpg       # Images des projets
```

## Déploiement sur GitHub Pages

1. Créer un dépôt GitHub nommé `<votre-pseudo>.github.io` (pour une URL du type `https://votre-pseudo.github.io`)
   **ou** un dépôt quelconque et activer GitHub Pages dans les paramètres du dépôt.
2. Pousser tous les fichiers à la racine du dépôt.
3. Dans **Settings → Pages**, choisir la branche `main` et le dossier `/root`.
4. Le site sera accessible en quelques minutes.

## Technologies utilisées

- Aucun framework JS requis
